# 开源

### **开源参数说明**

* `Optimization`: 是否开启了优化器, 开启后可生成较为精简的汇编代码
* `Runs`: 指定部署代码的每个操作码将在合同的整个生命周期中执行的大致频率
* `Solidity Viersion`: 编译器版本
* `License`: 开源协议
